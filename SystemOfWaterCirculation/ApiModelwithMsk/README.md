# System Of Water Circulation
**开放绿洲·水循环系统**  

本单元主要解决研究范围内的水循环过程，包括降雨模型、产汇流模型、下渗与蒸发模型、  
（一维、二维、三维）地表径流模型、地下径流模型、排水管网模型、潮汐模型、风场模型等；  
相关的衍生课题，包括洪水淹没风险评估、水资源联合调度、水库联合调度、闸泵联合调度等，
以及水体水质问题，河流泥沙问题，管道淤积问题等。
